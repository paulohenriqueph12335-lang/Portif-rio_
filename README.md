# Portif-rio_
Interactive business portfolio focused on Project Management, Business Intelligence, Data Analytics, Process Automation, and AI. Featuring real-world case studies, dashboards, and scalable solutions that transform complex challenges into measurable business results.


## Espaço de portfólio

Este repositório agora inclui uma landing page estática em `index.html`, com estilos em `styles.css`, para apresentar projetos, competências, indicadores e contato profissional.

### Prévia visual

![Prévia do espaço de portfólio](assets/portfolio-preview.svg)

A prévia acima fica versionada no repositório para evitar dependência de downloads externos durante a validação visual. Se o ambiente bloquear npm, apt ou pip com `403 Forbidden`, ainda é possível revisar o layout abrindo `index.html` no navegador ou visualizando o SVG em `assets/portfolio-preview.svg`.

Para regenerar a cópia de conferência da prévia sem instalar pacotes, execute:

```bash
node scripts/create-preview.js
```
